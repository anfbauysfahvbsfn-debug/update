# Core Steam Generator

##### This is a simple tool for generating Steam accounts.



##### Requirements:



* ###### Python 3.13 or higher
* ###### Google Chrome (for Selenium WebDriver)





##### Recommended:



* ###### A VPN (like Proton VPN or another free service) to enhance privacy and avoid IP restrictions during account creation.
* ###### A Mailticking account so you can login after creating an account and save the email for future logins.
